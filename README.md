# Project Backend

This is an IoT project. Here is the backend development. 
The Frontend repository is [here](https://github.com/linamariaum/jalian-planter-frontend).

This is a maven project generated with Java 8, Spring Boot Framework 2.2.3 and packaging jar

## Development server

Use your favorite IDE (Eclipse or IntelliJ), install maven dependencies and run `PlanterApplication.java` as Java Application on `http://localhost:8080/`.

Aditionally, you need RabbitMQ message broker working on port `15672` and a schema named `jalian-planter` with the following tables:
